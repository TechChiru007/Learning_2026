Day 1:- we are going to learn about cypress and its setup today

Setup requirements and process

1.NodeJs should be present in your system.
   cmd :- npm -version
2.Vs code IDE should be present .
3.Create a folder 
   mkdir folder name.
4.Create/initisalize a jsonfile.
   npm init -y
5.Open created folder in vs code.
6.Goto terminal install cypress
   npm install cypress --save -dev
7.Open cypress using 
   npx cypress open 
8. For reporting use mochaawesome dependency.
   npm i --save-dev cypress-mochawesome-reporter
   link :-https://www.npmjs.com/package/cypress-mochawesome-reporter
9. run cypress
   npx cypress run
   npx cypress run --reporter mochawesome

describe('User Authentication Suite', () => {

  // Runs once before any tests in this block
  before(() => {
    cy.log('Setting up global test data or seeding database');
  });

  // Runs before every single 'it' block
  beforeEach(() => {
    cy.visit('/login'); // Ensure we start at the login page every time
  });

  // Cleanup after every single 'it' block
  afterEach(() => {
    cy.log('Clearing local storage or cookies');
    cy.clearLocalStorage();
  });

  // Runs once after all tests in this block are finished
  after(() => {
    cy.log('All tests finished. Tearing down environment.');
  });

  it('should successfully log in with valid credentials', () => {
    cy.get('#username').type('standard_user');
    cy.get('#password').type('secret_password');
    cy.get('#login-button').click();
    cy.url().should('include', '/dashboard');
  });

  it('should display an error for invalid credentials', () => {
    cy.get('#username').type('wrong_user');
    cy.get('#password').type('wrong_password');
    cy.get('#login-button').click();
    cy.get('.error-message').should('be.visible');
  });

});
